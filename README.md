学生成绩管理系统（C语言版）

**面向班级的个性化成绩管理工具**，支持增删改查、成绩排名、班级筛选与数据持久化。

项目简介

本项目是基于 **C语言开发的模块化学生成绩管理系统**，专为班级成绩统计场景设计。相比基础版本，新增了 **班级分组、成绩等级评定、自动排名** 等实用功能，控制台交互简洁美观，数据支持本地持久化存储。  
适合计算机专业初学者巩固 C语言基础（结构体、文件IO、模块化编程），可直接作为实习简历项目。

核心功能
功能               描述
学生信息管理       支持添加、查询、修改、删除学生信息（学号、姓名、班级、三门科目成绩）

成绩自动处理       自动计算总成绩，根据总分评定 A/B/C/D 等级

班级筛选           按班级名称筛选学生，快速查看指定班级成绩情况

成绩排名           按总成绩降序排序并展示名次，直观呈现班级成绩梯队

数据持久化         通过二进制文件存储数据，程序重启后数据不丢失

技术亮点

1. **模块化设计**  
   将代码拆分为 main.c（主逻辑）、student.h（头文件）、student.c（功能实现），符合企业级 C语言开发规范。

2. **结构化数据存储**  
   使用结构体封装学生信息，支持多字段扩展。

3. **跨平台兼容性**  
   控制台输出格式兼容 Windows/Linux 系统。

4. **健壮性处理**  
   添加边界判断（如学生数量上限、成绩合理性校验）。

编译与运行

环境要求

- C语言编译器（GCC/Clang/MinGW）
- 任意控制台终端

编译命令

Linux/MacOS
gcc main.c student.c -o student_manage_custom

Windows（MinGW）
gcc main.c student.c -o student_manage_custom.exe

运行命令

Linux/MacOS
./student_manage_custom

Windows
student_manage_custom.exe

使用说明

1. 运行程序后，进入个性化班级系统界面。
2. 根据菜单提示输入操作序号：
   - **选择 1**：添加学生信息（需输入学号、姓名、班级、三门科目成绩）
   - **选择 5**：查看成绩排名，系统自动按总分降序排列
   - **选择 6**：按班级筛选学生
   - **选择 7**：保存数据并退出（数据会保存到 students.dat 文件）

项目结构

├── main.c          # 主程序入口，菜单逻辑处理
├── student.h       # 结构体定义与函数声明
├── student.c       # 核心功能实现（增删改查、排序、文件IO）
└── students.dat    # 数据持久化文件（运行后自动生成）

个性化定制

- 修改 main.c 中的 className 变量，替换为自己的班级名称。
- 调整 student.c 中 setGrade 函数的等级划分标准。
- 扩展结构体字段（如添加性别、年龄等信息）。
