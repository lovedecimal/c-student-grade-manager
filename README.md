**学生成绩管理系统（C语言版）**  
  ![License](https://img.shields.io/badge/License-MIT-green.svg)
![C Standard](https://img.shields.io/badge/C99%2B-Compliant-blue.svg)
这是个给班级用的轻量化成绩管理工具，能加删改查学生信息、自动算成绩排名和等级，还能按班级筛选，数据存在本地不会丢，特别适合计算机刚入门的同学练手，用来做课程设计或者写进简历都合适。  

项目简介  
这个系统是用C语言写的，主要给中小学、高校的班级统计成绩用，比基础版多了几个实用功能：  
- 能按班级来筛选、管理学生的成绩  
- 自动算学生总分，还能按分数评A/B/C/D等级（等级规则自己能改）  
- 控制台操作起来很顺手，数据存在本地文件里，关掉程序再打开也不会丢  

新手用这个练手，能把结构体、文件读写、模块化编程、排序算法这些C语言基础知识点都巩固一遍。  

核心功能  
功能分类           具体描述
学生信息管理       能加、能查、能改、能删学生信息，包括学号、姓名、班级，还有三门科目的成绩

成绩自动处理       自动算总成绩，按总分区间评A/B/C/D等级，等级的分数标准可以自己调

班级精准筛选       输入班级名，就能只显示这个班所有学生的成绩信息

成绩排名展示       按总成绩从高到低排好序，能清楚看到每个学生的名次和成绩梯队

本地数据持久化     数据存在students.dat这个二进制文件里，程序重启后数据还在

技术亮点  
1. **模块化设计**：把代码拆成了main.c（管主菜单和跟用户交互）、student.h（定义结构体和函数）、student.c（实现增删改查这些核心功能），符合平时企业里写C语言代码的规范；  
2. **结构化存数据**：用结构体把学生的各项信息包起来，想加性别、年龄这些字段也很方便；  
3. **跨平台能用**：控制台的输出格式在Windows、Linux、MacOS上都能正常显示；  
4. **防程序崩溃**：加了限制，比如学生数量有上限、成绩只能输0-100，避免输错东西导致程序卡死。  

编译与运行  
环境要求  
- 有C语言编译器就行，比如GCC、Clang、MinGW，推荐用GCC 4.8及以上版本  
- 随便一个控制台终端都能跑，比如Windows的CMD、PowerShell，或者Linux/Mac的终端  

编译命令  
Linux/MacOS  
gcc main.c student.c -o student_manage -std=c99  

Windows（MinGW）  
gcc main.c student.c -o student_manage.exe -std=c99  

运行命令  
Linux/MacOS  
./student_manage  

Windows  
student_manage.exe  

使用说明  
把编译好的文件运行起来，就能看到系统的主菜单；照着控制台的提示输操作序号就行，核心操作说一下：  
- 输1：添加学生信息，要依次填学号、姓名、班级，还有三门科目的成绩；  
- 输5：看全年级的成绩排名，会自动按总分从高到低显示名次；  
- 输6：按班级筛选学生，输个班级名，就只显示这个班的数据；  
- 输7：保存数据然后退出，数据会自动写到 students.dat 文件里。

个性化定制  
- 改默认班级：打开 main.c，找到 className 这个变量，换成你想设置的班级名就行；  
- 调成绩等级规则：改 student.c 里 setGrade 函数的分数区间，比如想把 A 等的分数线从 90 调到 85，改一下数值就好；  
- 加学生信息字段：在 student.h 的 Student 结构体里加新字段，比如 gender（性别）、age（年龄），然后把增删改查的逻辑同步改一下；  
- 加科目 / 成绩维度：给结构体里加新的成绩字段，再改一改总分的计算逻辑就行。  

注意事项  
- 成绩只能输 0-100 之间的数，输错了会提示你重新输；  
- students.dat 文件会生成在程序运行的文件夹里，删了这个文件，存的学生数据就全没了；  
- 要是编译的时候报错，记得确认编译器支持 C99 及以上标准，编译命令里加上 -std=c99 就行。
